Document Object Model

Another Built-in Object. It represents / models the webpage. Document object is linked to the webpage, so if we change a property of the object, then it will be displayed to the page.

We can have HTML elements inside Javascript!!! The DOM object combines JS and HTML together, and it gives JS to control the webpage.

Note: An html elemnt inside JS, is just a JavaScript object, so it has its own properties, like document.body is an object, and one of its properties is document.body.innerHTML

Methods of document:

  • document.quarySelector(): It lets us get ANY ELEMNT from the page and put it inside JS.
    input: string that is the html elemnt, like 'body', 'button', 'h1' and more.
  • Benefits: Getting an HTML element from the page, put it inside JS, an dthen save it in a variable.
  • DOM gives JS full control of the webpage
