AutoBox object

When JS wrap something like an object, we call it AutoBoxing, just like when using methods on strings, JS consider the string as a special object.

Autoboxing works for: String, Numbers, and Bool

Objects are refrences.

Refrences has intresting features. As you can see you can change the properties of an object even if we have const object = ... .
That is because const is refrenced to the adress, and the adress is always constant, but we can chaneg the value.

Comparing Objects: If we want to compare objects, we should just make a function that chekcs if all the properties are the same. If we do obect1 === object2, we are basically checking if their refrences are the same or not.
